**Once you select "Android", the form will change:**

1. **Name:** `Fillora Android Client`
2. **Package name:** `com.fillora.app`
3. **SHA-1 certificate fingerprint:** Paste this:
   ```
   28:70:EA:81:48:9A:76:47:72:B2:58:75:CD:60:2D:86:5D:C3:8B:81
   ```
4. **Click "CREATE"**
